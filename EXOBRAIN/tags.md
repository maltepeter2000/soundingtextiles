    description: >
      ExoBrain is a touch-sensitive textile interface that conveys the sensation of "brainrot"
      from overconsumption of internet content. With the help of interactive internet meme sounds,
      ExoBrain reflects the overload of digital stimuli and its impact on the mind.

      ExoBrain project includes:
      - A cap as the main wearable element
      - Bela Mini and Trill for responsive interaction
      - Thick, conductive threads shaped like brain folds

      Using Bela and Pure Data, the project aims to use meme sounds with touch triggers
      spatialized according to the touch position.
